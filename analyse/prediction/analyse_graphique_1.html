Problématique

Ce graphique montre les prédictions des locations de vélos du programme Santander Cycle Hire à Londres, en utilisant un modèle combiné ARIMA + XGBoost. L'objectif est de prévoir les tendances futures des locations de vélos, d'évaluer la performance du modèle et de comprendre les variations saisonnières et les impacts des événements majeurs (comme la pandémie de COVID-19).

Origine des données

Les données proviennent du site officiel de la ville de Londres, data.london.gov.uk. Elles incluent le nombre total de locations de vélos par jour, mois et année, depuis le lancement du programme le 30 juillet 2010.

Traitement des données

  • Les données ont été extraites à partir d'un fichier CSV contenant les locations de vélos par jour.
  • La colonne de date a été convertie en format datetime pour permettre une analyse temporelle précise.
  • La colonne du nombre de locations a été nettoyée pour supprimer les espaces insécables et autres caractères indésirables, puis convertie en nombres entiers.
  • Les données ont été agrégées par mois pour observer les tendances mensuelles.
  • Les données ont été divisées en un ensemble d'entraînement (avant 2023) et un ensemble de test (après 2023).

Méthodologie statistique

  • Modèle ARIMA : Un modèle ARIMA (AutoRegressive Integrated Moving Average) a été utilisé pour capturer les tendances temporelles et les composantes saisonnières des données. Les paramètres (p, d, q) ont été sélectionnés automatiquement.
  • Modèle XGBoost : Un modèle XGBoost (Extreme Gradient Boosting) a été utilisé pour corriger les erreurs du modèle ARIMA et améliorer la précision des prédictions. Il a été entraîné sur les résidus du modèle ARIMA.
  • Combinaison des modèles : Les prédictions finales sont une combinaison des prédictions ARIMA et des ajustements apportés par XGBoost.

Justification des choix

  • Le modèle ARIMA est bien adapté pour les séries temporelles avec des tendances et des saisonnalités.
  • XGBoost est utilisé pour capturer les relations non linéaires et améliorer la précision des prédictions.
  • La combinaison des deux modèles permet de tirer parti des forces de chaque approche.

Observations préliminaires

  • Le modèle ARIMA capture bien les tendances générales et les variations saisonnières, mais il peut sous-estimer ou surestimer certaines valeurs.
  • XGBoost améliore les prédictions en ajustant les erreurs du modèle ARIMA, en particulier pour les variations plus complexes.
  • Les prédictions combinées (ARIMA + XGBoost) montrent une bonne adéquation avec les données réelles, en particulier pour les mois récents.
  • Une augmentation progressive des locations de vélos est prédite pour les mois à venir, reflétant une reprise post-pandémique et une popularité croissante du vélo comme mode de transport.

Informations supplémentaires

  • Performance du modèle : Le modèle combiné ARIMA + XGBoost montre une bonne précision sur les données de test, avec une erreur moyenne faible.
  • Limites des données : Les prédictions sont basées sur des données historiques et peuvent être affectées par des événements imprévus (comme des changements politiques ou des conditions météorologiques extrêmes).
  • Utilisation des prédictions : Ces prédictions peuvent être utilisées pour planifier les ressources, anticiper les pics de demande et évaluer l'impact des politiques de transport.

Utilisation du graphique
